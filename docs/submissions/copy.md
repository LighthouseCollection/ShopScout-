# ShopScout Store Listing Copy

## Short Description

Collect products from shopping pages, organize them into comparison lists, normalize specs, and prepare AI-assisted buying analysis.

## Full Description

ShopScout is a private product comparison workspace for people who research purchases across Amazon, eBay, Walmart, Best Buy, Target, Costco, Newegg, Home Depot, Lowe's, AliExpress, Alibaba, Temu, SHEIN, and other product pages.

Use the browser side panel to capture the current product, add products from open tabs, or paste product URLs into a selected list. Open the comparison dashboard to sort, filter, group, compare, normalize, and export the collected products.

ShopScout focuses on structured buying decisions:

- Capture product names, URLs, prices, brands, identifiers, ratings, seller details, images, feature bullets, and technical specs.
- Normalize messy product data such as duplicate field names, unit formats, colors, sizes, and common aliases.
- Compare products in a spreadsheet-style dashboard with product rows, comparison views, filters, grouping, and editable fields.
- Use Manual AI prompts or connected AI providers to verify claims, fill missing specs, compare price and value, and produce buying recommendations.
- Paste Manual AI results back into ShopScout so corrected facts can update the comparison table.

ShopScout is not a coupon injector, affiliate redirector, price tracker, or background scraper. Capture is user-initiated and list-based.

## Key Features

- One-click product capture from supported shopping pages.
- Add all product tabs in the current browser window.
- Structured comparison dashboard with AG Grid.
- Product-specific technical specification normalization.
- Duplicate candidate detection.
- Category and vertical-aware normalization libraries.
- Manual AI prompt workflow for users who do not want to store API keys.
- Optional connected AI providers for automatic analysis.
- Export collected products and AI analysis to common formats.

## Privacy Summary

ShopScout stores captured product lists locally in the browser extension storage and IndexedDB on this machine. Product data is only sent to an AI provider when the user explicitly runs an AI workflow. API keys, when configured, are stored in local browser extension storage and can be removed from Settings.
