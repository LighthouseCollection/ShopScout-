# ShopScout Privacy Policy

Last updated: 2026-07-20

ShopScout is a browser extension for collecting, organizing, normalizing, and comparing product information from shopping pages.

## Data ShopScout Collects

When the user chooses to capture a product, ShopScout may save product-related information from the active page, including:

- Product name
- Product URL
- Price
- Brand
- Model and identifiers
- Seller and source
- Rating and review count
- Product images
- Feature bullets
- Technical specifications
- User notes
- AI analysis results pasted or generated by the user

ShopScout does not intentionally collect passwords, payment information, private messages, or unrelated browsing history.

## Where Data Is Stored

Captured product data is stored locally on the user's device using browser extension storage and IndexedDB.

ShopScout does not operate a ShopScout server for collecting product lists, browsing history, or personal data.

## AI Providers

If the user connects an AI provider or uses a Manual AI workflow, selected product facts may be sent to that provider. The provider's own privacy policy and terms apply to that interaction.

ShopScout only sends AI prompts when the user explicitly starts an AI workflow. Users can choose compact prompts to reduce the amount of product data included.

## API Keys

When users enter API keys, ShopScout stores them in local browser extension storage on the user's device. Browser extension storage is convenient but is not a password vault. Users can remove keys from Settings at any time.

## Data Sharing

ShopScout does not sell captured product data. ShopScout does not inject affiliate links or redirect purchases.

## User Control

Users can delete products, delete lists, remove AI keys, export data, or uninstall the extension to remove extension-managed local data according to browser behavior.
