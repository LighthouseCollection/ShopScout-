# ShopScout Third-Party Notices

This document summarizes third-party materials and libraries used by ShopScout. It is not a substitute for the full license text in each upstream project.

## Runtime Libraries

- AG Grid Community: used for dashboard grid rendering.
- Dexie: used for IndexedDB access.
- Browser extension APIs provided by Chromium, Microsoft Edge, and Firefox.

## Data and Vocabulary Sources

- Shopify Product Taxonomy: category and taxonomy reference data.
- Google Product Taxonomy: category reference data.
- GS1 GPC subset: product-classification reference data.
- Icecat-derived generated libraries: product feature/category normalization support.
- Amazon ESCI dataset-derived fixtures/generators: substitute-signal experimentation and product-search relevance research.
- Schema.org generated reference data: structured vocabulary alignment.

## Bundled Generated Data

ShopScout ships generated JSON libraries used by the normalization pipeline. Raw source corpora are not shipped in the extension package.

## Attribution

See the repository `NOTICE` file and generated library manifests for source, license, hash, and build provenance.
